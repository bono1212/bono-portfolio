퀵 에디터에는 4가지가 있으면 단축키는 모두 Ctrl + E

1.Emmet
  특별한 문법으로 코드를 쉽게 자동완성 해줍니다.
  HTML, CSS 작업시 없어서 안될 필수 확장 기능이다.
  http://docs.emmet.io/abbreviations/syntax/
  https://opentutorials.org/course/128/1395

1-1.Reverse Emmet (ctrl+shift+e)
   가끔 html로 작업하다 반복이 필요할 때 아 이건  Emmet 으로 10번 반복하면 금방인데.. 
   그럼 다시 Emmet으로 바꾸고 *10 하면 되지 않을까 해서 찾은 확장 기능입니다. 
   말그대로 HTML 태그를  emmet 코드로 변환해 줍니다.

2.beautify.io
  코드를 상당히 깔끔하게 정렬해줍니다.
  JS, HTML, CSS 코드를 포멧에 맞게 자동 맞춤 기능

3.Live Preview for Responsive
  변경된 코드를 브라우저에서 실시간으로 보여줍니다 
  javascript를 포함해서요! 굉장합니다 

4.JS Live Debugger 
   자바스크립트의 문법을 체크해주고 디버깅을 해줍니다.

5.eqFTP
  서버에 직접적으로 실시간 파일 수정, 삭제, 생성이 가능해요

6.Todo
  Display all TODO comments in current document or project.

7.HTML Wrapper

8.CSSFier

9.Custom Work
  brackets 의 탭기능을 사용하게 해줍니다.
  열려있는 각 파일간을 tab으로 이동

10.Brackets Power Mode

11.Minifier
   JS, CSS 파일을 압축하여 파일 이름에 .min 붙여 저장하는 기능

12.Indent Guide
   인덴트 가이드 라인(세로 선)을 표시해 주는 기능

13.Quick Search
   파일 중에 선택한 키워드와 같은 문자열을 하이라이팅(강조) 표시하는 기능.

14.Color Picker
   컬러 코드에서 Ctrl + E 햇을 때 나오는 색상 지정기를 그랙피컬 하게 표시하는 기능 

15.Bracket Snippets(by edc)
  반복 사용되는 코드를 스니핏으로 등록하여 사용할 수 있게 해준다

16.Bracket File Icons 
  파일 아이콘 테마

17.Bracket Tree icons 
  파일 , 폴더 표시 테마

18.Interactive Linter 
각종 코드 오류 체크 확인 기능

19.Visual Studio Dark
테마-설치안해도됨
